Contains the logs concerning the initialization data update and error.

Structure:
    \update - The update logs based on the date. Contains data about update and error.
    \error  - Contains only the logs of the error concerning The initialization data.

Record info:
    level       - The message level.
    message     - The message given.
    timestamp   - When the message was created.
    err_message - The error message given when an error occurs.
    data        - Data that might throw an error.
